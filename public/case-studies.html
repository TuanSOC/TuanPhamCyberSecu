SOC / Blue Team Case Studies

Evidence-first project writeups.

These case studies are written like analyst notes: problem, environment, signal, action, result, and lesson. They are designed to help recruiters and technical reviewers scan the work quickly.

Case Study 01 / WAF

Barracuda WAF Security Lab

Problem

Validate simulated web attacks behind a WAF instead of trusting block counts only.

Environment

DVWA, Barracuda WAF, SMTP alerting, on-prem style lab.

Signal

Rule hits, suspicious request paths, source behavior, upload/XSS attempts.

Action

Tuned WAF policies, generated attack traffic, reviewed events, built Python SMTP alerts.

Result

Repeatable WAF tuning and evidence collection workflow.

Learning: A useful alert needs request context, rule triggered, source behavior, and a recommended next action.

Case Study 02 / Detection Engineering

AI-based Unsupervised Threat Detection

Problem

Detect brute-force and SQLi-like behavior when labeled attack data is limited.

Environment

Wazuh, pfSense, Python, Scikit-learn, Isolation Forest, Autoencoder.

Signal

Authentication patterns, HTTP features, Wazuh alerts, firewall telemetry.

Action

Normalized events, engineered features, compared anomaly scores, reviewed suspicious windows.

Result

Proof of concept for surfacing suspicious authentication and SQLi patterns.

Learning: ML detection is stronger when the model output is paired with analyst-readable context.

Case Study 03 / Hardening

Oracle SQL Database Security

Problem

Reduce risk from excessive privileges, weak auditing, and SQL injection test scenarios.

Environment

Oracle SQL, role-based access, audit logs, SQL injection test cases.

Signal

User privileges, query activity, audit records, unauthorized access attempts.

Action

Applied least privilege, enabled auditing, tested injection scenarios, documented hardening actions.

Result

Clearer access boundaries and improved audit visibility.

Learning: Database hardening is strongest when access design, query testing, and audit review are treated together.
